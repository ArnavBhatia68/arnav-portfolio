# Arnav Portfolio Starter (Next.js + Tailwind)

## Quickstart
1) Install Node LTS (recommended 20.x) via nvm:
   ```bash
   nvm install --lts
   nvm use --lts
   ```
2) Install deps and run:
   ```bash
   npm install
   npm run dev
   ```
3) Open http://localhost:3000

## Customize
- Edit text in `src/app/page.tsx` (hero) and `src/app/about/page.tsx`.
- Add/update projects in `src/data/projects.ts`.
- Drop your resume as `public/Arnav_Bhatia_Resume.pdf`.
- Update email/links in `src/app/page.tsx` and `src/app/contact/page.tsx`.

## Deploy
- Push to GitHub, then import on Vercel.
- Set your custom domain in Vercel Project → Settings → Domains.
